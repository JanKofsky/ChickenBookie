# Chicken Bookie Vercel Site

Static landing page for `chickenbookie.com`.

## Vercel setup

1. Import `JanKofsky/ChickenBookie` into Vercel.
2. Set the project root directory to `vercel-site`.
3. Framework preset: `Other` or static/default.
4. Build command: leave blank.
5. Output directory: leave blank.
6. Add `chickenbookie.com` in Vercel project settings under Domains.

The landing page currently links to `https://chickenbookie.streamlit.app`. If the Streamlit app URL is different, update both links in `index.html`.
